# MyBook
这个应用用了豆瓣的图书api。可以搜索豆瓣上面的图书并查看图书信息和图书相关的读书笔记，也可以使用条形码扫描图书搜索。还可以把某本图书或者某篇笔记加入收藏，方便查看。(该应用模仿的https://github.com/JayFang1993/ScanBook开源项目)
